Hakkında

Merhaba, Ben Necati Karabiberler. 2017 yılında Şehit Serhat Sığnak Mesleki ve Teknik Anadolu Lisesi'nden Bilişim Teknolojileri alanında eğitim aldım. Web tasarım dalında HTML, CSS, PHP, ve C# dillerinde eğitim aldım. Eğitimimin son yılında Oreon Yazılım Şirketi'nde staj yaptım.

Lise hayatımın sona ermesinin ardından kendimi geliştirmek için birçok kursa katıldım. 2023 yılının başında Neos Yazılım Akademisi'nde yazılım konusunda daha da gelişmek için eğitime başladım. Bu eğitim kapsamında Front-end kısmında HTML, CSS, Bootstrap, Javascript, React, JQUERY ve JQUERY UI dillerinde dersler aldım. Back-end kısmında ise PYTHON, DJANGO, NODE JS ve C# Eğitimi aldım.
